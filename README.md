## Environment setup

- python3 -m venv env
- pip install pandas
- pip install ipykernel
- pip install openpyxl
- python3 -m pip install --upgrade pip
- python3 -m pip install --upgrade Pillow
- pip install matplotlib
- pip freeze > requirements.txt
- pip install -r requirements.txt
